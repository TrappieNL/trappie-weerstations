# Trappie Meteo

Een klassiek WordPress-thema voor meteorologie, weerstations en weerstationtechniek. Het thema werkt zelfstandig en bevat extra styling voor de plugin **Trappie Weerstations**.

## Installatie

1. Upload `trappie-meteo.zip` via **Weergave > Thema's > Nieuw thema > Thema uploaden**.
2. Activeer **Trappie Meteo**.
3. Laat de plugin **Trappie Weerstations** actief voor het overzicht, de filters en detailpagina's.
4. Stel eventueel een menu in onder **Weergave > Menu's**. Zonder ingesteld menu gebruikt het thema een praktische standaardnavigatie.

## Startpagina

`front-page.php` is direct de visuele startpagina. De nieuwste drie gepubliceerde weerstations en de nieuwste drie gewone artikelen worden automatisch getoond.

## Afbeelding

Het meegeleverde hoofdbeeld is speciaal voor dit thema gegenereerd en bevat geen merken, tekst of externe beeldrechten.
